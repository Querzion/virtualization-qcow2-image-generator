# virtualization-qcow2-image-generator
This script creates a set of predetermined qcow2 images for virtual machines, 
they will be created in the /var/lib/libvirt/images folder which is the default read folder.

It's a script that is easy to read, add and remove from.

## Usage
```bash
chmod +x virtualization-qcow2-image-generator.version.03.sh
sudo ./virtualization-qcow2-image-generator.version.03.sh
```
